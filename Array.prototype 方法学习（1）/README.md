# Array.prototype 方法学习（1）

## 任务描述

使用 Array.prototype.method() 来完成挑战吧！在此挑战中，你能学会使用 Array 的原型方法来简化代码的书写，包括但不限于 filter()、map()、sort()、reduce() 等方法的学习和实践。

## 任务目的

### JavaScript

1. 熟练掌握 filter() 方法的使用
2. 熟练掌握 sort() 方法的使用
3. 熟练掌握 map() 方法的使用
4. 熟练掌握 reduce() 方法的使用
5. 掌握额外的例如 includes()、forEach()、slice() 等方法的学习
